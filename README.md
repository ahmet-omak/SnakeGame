# SnakeGame
**It's a Simple Snake Game**
# Qualifications
-**Snake is made of black squares**<br/>
-**Snake can go through the walls**<br/>
-**Snake gets bigger when it eats the red square**<br/>
-**Snake dies when it hits one of its parts**<br/>
